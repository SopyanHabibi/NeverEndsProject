# Neira

> **Not just another AI assistant. A companion that grows with you.**

Neira is a personal AI assistant built from scratch as a long-term project. Unlike traditional assistants that simply answer questions, Neira is designed to remember, adapt, and become more useful over time.

This project focuses on creating an AI that feels less like a chatbot and more like a digital companion capable of understanding its user through memory, scheduling, and continuous improvements.

---

# Why Neira?

Most AI assistants are stateless. They answer a question and immediately forget the conversation.

Neira is built with a different philosophy.

Instead of starting every conversation from zero, Neira gradually builds memories, organizes information, and becomes more personalized while still allowing the user to control what should or shouldn't be remembered.

The goal isn't to replace existing AI models.

The goal is to build the experience around them.
